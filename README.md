**Demountable menu integration plug-in based on vue.**



## Installation

npm:

`npm install --save vue-tmenu`

yarn:

`yarn add vue-tmenu`

## Default import

Install all the components:
```js
import Vue from 'vue'
import vtmenu from 'vue-tmenu'
Vue.use(vtmenu)
```
